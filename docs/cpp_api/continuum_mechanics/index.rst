====================
Continuum Mechanics
====================

The Continuum Mechanics module provides functions for tensor operations, strain/stress transformations, 
and kinematics in both small and finite strain contexts.

Overview
========

This module contains the core mechanical functions of Simcoon:

- **Functions**: Core tensor operations, strain measures, stress transformations
- **Material**: Material property definitions and conversions
- **Micromechanics**: Eshelby solutions and inclusion problems
