Overview
========

Functions for hyperelastic material models, including strain energy derivatives, 
stress computations, and tangent moduli for various hyperelastic models 
(Neo-Hookean, Mooney-Rivlin, Ogden, etc.). Supports both invariant-based 
and principal stretch-based formulations.
