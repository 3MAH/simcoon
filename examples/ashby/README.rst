Ashby Material Selection Examples
---------------------------------

Examples demonstrating the ``simcoon.ashby`` subpackage for material property
exploration and selection.

This gallery contains examples showing:

- **Material datasets** — Loading the built-in curated dataset and importing custom CSV data
- **Ashby diagrams** — Creating Young's modulus vs density charts with envelopes and guide lines
- **Simcoon bridge** — Converting materials to stiffness/compliance tensors for simulation
- **Materials Project** — Fetching and combining materials from online databases
